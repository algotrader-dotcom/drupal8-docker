# Drupal Dockerized
## How to run
```
git clone https://github.com/thuannvn/drupal8-docker.git
```
```
cd drupal8-docker
```
```
chmod +x start.sh
```
```
./start.sh

```

## Then acess http://127.0.0.1:8000 for Drupal Installation Process
database: drupal  
username: drupal  
password: drupal  
host: mariadb
